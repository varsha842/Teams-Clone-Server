# Video Conferening Server
## Signalling Server

This is a signalling server, implemented for WebRTC applications.

## Getting Started

### Prerequisite

- NodeJS (>=12+)

### Technologies used

- NodeJS
- Typescript
- Websocket


### Installation

- Clone this repo.
```sh
git clone https://github.com/varsha842/Teams-Clone-Server.git
```

- Install modules.
```sh
npm install
```

- Run the server on terminal.
```sh
npm start
```
